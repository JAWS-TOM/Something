# Vim 操作速查表
按「增删改查」「光标移动」「其他功能」分类，便于快速查阅。

## 一、增：插入编辑（普通模式 → 编辑模式）
| 命令 | 功能描述 | 适用场景 |
|------|----------|----------|
| `i`  | 光标当前位置前插入 | 补全单词中间内容 |
| `I`  | 当前行首非空字符前插入 | 代码行首加注释（//） |
| `a`  | 光标当前位置后插入 | 单词末尾补全 |
| `A`  | 当前行尾插入 | 没啥用 |
| `o`  | 当前行下方新建行并插入 | 向下扩展 |
| `O`  | 当前行上方新建行并插入 | 向上补充 |

## 二、删：精准删除（普通模式）
| 命令       | 功能描述 | 示例 |
|------------|----------|------|
| `x`        | 删除光标所在字符 | 删单个错误字符 |
| `dd`       | 删除整行 | *** |
| `数字+dd`  | 删除指定行数（含当前行） | `3dd` 删当前行及下方2行，删除全文也很好用 |
| `dw`       | 删除到下一个单词开头 | 删当前单词剩余部分 |
| `d$` / `d0`| 删除到行尾 / 行首 | 清空行尾/行首 |
| `df+字符`  | 删除到目标字符（含该字符） | `df-` 删到当前行第一个“-” |
| `dG` / `dgg`| 删除到文件结尾 / 开头 | 清空文件后半段/前半段 |

## 三、改：快速修改（普通模式，删后自动进编辑模式）
| 命令       | 功能描述 | 示例 |
|------------|----------|------|
| `cw`       | 修改当前单词 | 光标在“tes”按 `cw` 改“test” |
| `cc`       | 修改整行 | 清空当前行并直接编辑 |
| `数字+cc`  | 修改指定行数 | `2cc` 改当前行+下一行 |
| `c$` / `c0`| 修改到行尾 / 行首 | 重写行尾/行首内容 |
| `cf+字符`  | 修改到目标字符（含该字符） | `cf=` 改到当前行第一个“=” |

## 四、查：查找与替换（普通模式）
### 1. 查找
| 命令       | 功能描述 | 操作 |
|------------|----------|------|
| `/关键词`  | 正向查找（从光标向下） | 输 `/error` 找所有“error”，`n` 下一个，`N` 上一个 |
| `?关键词`  | 反向查找（从光标向上） | 输 `?success` 找所有“success”，`n` 上一个，`N` 下一个 |
| `*` / `#`  | 查找光标所在字符（下/上） | 光标在“name”按 `*` 找所有“name” |

### 2. 替换
| 命令格式               | 功能描述 | 示例 |
|------------------------|----------|------|
| `范围 s/旧内容/新内容/选项` | 批量替换 | - |
| `%s/旧/新/g`           | 全文件替换（不确认） | `%s/old/new/g` 全文件换“old”为“new” |
| `n,m s/旧/新/c`        | 指定行替换（需确认） | `5,10s/error/INFO/c` 5-10行换“error”为“INFO”，每处确认 |
| `s/旧/新/`             | 当前行仅替换第一个匹配 | 只改当前行第一个“old” |

## 五、光标移动（普通模式，告别方向键）
| 分类       | 命令       | 功能描述 | 示例 |
|------------|------------|----------|------|
| 基础移动   | `h/j/k/l`  | 左/下/上/右 | `5j` 向下跳5行 |
| 行内移动   | `0` / `^` / `$` | 行首/行首非空字符/行尾 | `^` 直接到代码行开头 |
| 单词移动   | `w` / `b` / `e` | 下一个/上一个单词开头/当前单词结尾 | `3w` 跳3个单词 |
| 段落移动   | `{` / `}`  | 上一个/下一个段落开头 | 长文档快速跳段落 |
| 全局移动   | `gg` / `G` | 文件开头 / 文件结尾 | 一键到顶/到底 |
| 全局移动   | `数字+G`   | 跳转到指定行 | `100G` 到第100行 |
| 历史定位   | `Ctrl+o` / `Ctrl+i` | 上一个/下一个光标位置 | 从第5行跳100行后，`Ctrl+o` 回第5行 |

## 六、其他实用功能（普通模式）
| 功能       | 命令       | 操作说明 |
|------------|------------|----------|
| 复制       | `yy` / `数字+yy` | 复制整行 / 指定行数 |
| 复制       | `yw` / `y$` / `y0` | 复制单词 / 到行尾 / 到行首 |
| 粘贴       | `p` / `P` | 粘贴到光标后 / 光标前 |
| 可视化选择 | `v` / `Shift+v` / `Ctrl+v` | 字符可视化 / 行可视化 / 块可视化 |
| 撤销/重做  | `u` / `Ctrl+r` | 撤销上一步 / 重做被撤销操作 |
| 保存退出   | `:w` / `:q` / `:wq` / `:q!` | 保存 / 退出（未改） / 保存退出 / 强制退出（不保存） |
| 分屏操作   | `:split 文件名` / `:vsplit 文件名` | 横向分屏打开文件 / 纵向分屏打开文件 这里可以Ctrl +w 再按=平分两个窗口|
